// FAQ

Common Questions

Is TransFormX really free?
+
Yes, completely. All core conversion and download features are free with no file count limits. We offer optional Pro plans for commercial API usage and priority processing, but personal use is always free.
What is the maximum file size?
+
Free accounts can convert files up to 2 GB. Pro accounts support files up to 50 GB. For video downloads, there's no practical size limit as files are streamed directly.
Are my files kept private?
+
Absolutely. Files are transferred over encrypted HTTPS, processed in isolated containers, and deleted from our servers within 60 minutes. We never read, analyze, or share your file contents.
Why is my download URL not working?
+
Some platforms frequently change their APIs to prevent downloading. We update our systems regularly. If a platform isn't working, try again in a few minutes or check our status page. Private/DRM-protected content cannot be downloaded.
Can I convert files on my phone?
+
Yes! TransFormX is fully responsive and works on iOS and Android browsers. You can even install it as a PWA from your browser menu for an app-like experience with offline support.
Do you have an API?
+
Yes. Our REST API is available for developers. Sign up for a free API key on the developer portal. Rate limits apply to free keys; Pro keys have no limits.
